🚗 Traffic Dodger

A simple lane-dodging game built with React Native and Expo.

Switch lanes to dodge oncoming traffic and survive as long as possible!

Gameplay


Car sits on a 3-lane road
Move Left / Move Right buttons to switch lanes
Dodge oncoming cars and trucks to increase your score
Crashing ends the game and shows your final score
High score is saved locally


Tech Stack


React Native (Expo)
AsyncStorage for high score


Run Locally

bashgit clone https://github.com/YOUR_USERNAME/TrafficDodger_Game.git
cd TrafficDodger_Game
npm install
npx expo start

Scan the QR code with the Expo Go app to play.
